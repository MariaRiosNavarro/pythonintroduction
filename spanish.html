Introducción a Python

¿Qué es Python?

Python es un lenguaje de programación fácil de aprender y poderoso. A menudo se utiliza en áreas como el análisis de datos, el desarrollo web y el aprendizaje automático. Python tiene una sintaxis clara y legible que hace que sea fácil escribir y mantener aplicaciones complejas.

¿Por qué debo aprender Python?

  • Python es uno de los lenguajes de programación más populares y es utilizado por muchas empresas en todo el mundo.
  • Hay una gran y activa comunidad dispuesta a ayudar a los nuevos usuarios.
  • Python tiene muchas bibliotecas y herramientas predefinidas que facilitan el desarrollo de aplicaciones.

¿Cómo se programa en Python?

Para programar en Python, primero debe configurar un entorno de desarrollo adecuado. Hay muchas opciones, incluyendo entornos de desarrollo integrados (IDEs) como PyCharm y Visual Studio Code, o editores de texto simples como Notepad++ o Sublime Text. Una vez que haya configurado un entorno de desarrollo, puede comenzar a escribir código Python.

En Python hay diferentes tipos de datos, como enteros (integers), números de punto flotante (floats) y cadenas (strings). Aquí hay un ejemplo de cómo usar estos tipos de datos en Python:

  # Entero (integer)
  edad = 30
  
  # Número de punto flotante (float)
  pi = 3.14159
  
  # Cadena (string)
  nombre = "Juan"

En Python también hay diferentes operadores que puede usar para realizar cálculos y evaluar condiciones. Aquí hay algunos ejemplos de operadores en Python:

        # Operadores aritméticos
        a = 5
        b = 3
        
        # Adición
        c = a + b
        print(c)  # Salida: 8
        
        # Sustracción
        c = a - b
        print(c)  # Salida: 2
        
        # Multiplicación
        c = a * b
        print(c)  # Salida: 15
        
        # División
        c = a / b
        print(c)  # Salida: 1.6666666666666667
        
        # Operadores de comparación
        a = 5
        b = 3
        
        # Igualdad
        c = a == b
        print(c)  # Salida: False
        
        # Desigualdad
        c = a != b
        print(c)  # Salida: True
        
        # Mayor que
        c = a > b
        print(c)  # Salida: True
        
        # Mayor o igual que
        c = a >= b
        print(c)  # Salida: True
        
        # Menor que
        c = a < b
        print(c)  # Salida: False
        
        # Menor o igual que
        c = a <= b
        print(c)  # Salida: False
        
        # Operadores lógicos
        a = True
        b = False
        
        # Y lógico
        print(a and b)  # Salida: False
        
        # O lógico
        print(a or b)  # Salida: True
        
        # Negación lógica
        print(not a)  # Salida: False

En Python también hay diferentes estructuras de control que puede usar para controlar el flujo de su programa. Aquí hay algunos ejemplos de estructuras de control en Python: 

        # Estructuras de control de flujo
        # Si
        x = 5
        y = 3
        if x > y:
          print("x es mayor que y")
        
        # Si-sino
        x = 3
        y = 3
        if x > y:
          print("x es mayor que y")
        else:
          print("x no es mayor que y")
        
        # Si-sino-si
        x = 3
        y = 5
        if x > y:
          print("x es mayor que y")
        elif x < y:
          print("x es menor que y")
        else:
          print("x es igual a y")
        
        # Bucle for
        for i in range(5):
          print(i)  # Salida: 0 1 2 3 4
        
        # Bucle while
        i = 0
        while i < 5:
          print(i)
          i += 1  # Incrementar i en 1
        
        # Funciones
        def decir_hola(nombre):
          print("Hola, " + nombre + "!")
        
        decir_hola("Juan")  # Salida: Hola, Juan!
