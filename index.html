Einführung in Python

Was ist Python?

Python ist eine leicht zu lernende und mächtige Programmiersprache. Sie wird häufig in Bereichen wie Datenanalyse, Webentwicklung und maschinellem Lernen verwendet. Python hat eine klare und lesbare Syntax, die es einfach macht, komplexe Anwendungen zu schreiben und zu warten.

Warum sollte ich Python lernen?

  • Python ist eine der beliebtesten Programmiersprachen und wird von vielen Unternehmen weltweit verwendet.
  • Es gibt eine große und aktive Community, die bereit ist, Neueinsteigern zu helfen.
  • Python hat viele vorgefertigte Bibliotheken und Werkzeuge, die die Entwicklung von Anwendungen erleichtern.

Wie programmiert man in Python?

Um in Python zu programmieren, müssen Sie zunächst eine geeignete Entwicklungsumgebung einrichten. Es gibt viele Optionen, darunter integrierte Entwicklungsumgebungen (IDEs) wie PyCharm und Visual Studio Code, oder einfache Texteditor wie Notepad++ oder Sublime Text. Sobald Sie eine Entwicklungsumgebung eingerichtet haben, können Sie mit dem Schreiben von Python-Code beginnen.

In Python gibt es verschiedene Datentypen, wie zum Beispiel Ganzzahlen (integers), Fließkommazahlen (floats) und Zeichenketten (strings). Hier ist ein Beispiel, wie Sie diese Datentypen in Python verwenden können:

      # Ganzzahl (integer)
      age = 30
      
      # Fließkommazahl (float)
      pi = 3.14159
      
      # Zeichenkette (string)
      name = "John"

In Python gibt es auch verschiedene Operatoren, die Sie verwenden können, um Berechnungen durchzuführen und Bedingungen zu überprüfen. Hier sind einige Beispiele für Operatoren in Python:

            # Arithmetische Operatoren
            a = 5
            b = 3
            
            # Addition
            c = a + b
            print(c)  # Ausgabe: 8
            
            # Subtraktion
            c = a - b
            print(c)  # Ausgabe: 2
            
            # Multiplikation
            c = a * b
            print(c)  # Ausgabe: 15
            
            # Division
            c = a / b
            print(c)  # Ausgabe: 1.6666666666666667
            
            # Modulo
            c = a % b
            print(c)  # Ausgabe: 2
            
            # Vergleichsoperatoren
            x = 5
            y = 3
            
            # Gleich
            print(x == y)  # Ausgabe: False
            
            # Nicht gleich
            print(x != y)  # Ausgabe: True
            
            # Größer als
            print(x > y)  # Ausgabe: True
            
            # Größer als oder gleich
            print(x >= y)  # Ausgabe: True
            
            # Kleiner als
            print(x < y)  # Ausgabe: False
            
            # Kleiner als oder gleich
            print(x <= y)  # Ausgabe: False
            
            # Logische Operatoren
            a = True
            b = False
            
            # Und
            print(a and b)  # Ausgabe: False
            
            # Oder
            print(a or b)  # Ausgabe: True
            
            # Nicht
            print(not a)  # Ausgabe: False

In Python gibt es auch verschiedene Steuerungsstrukturen, die Sie verwenden können, um den Fluss Ihres Programms zu steuern. Hier sind einige Beispiele für Steuerungsstrukturen in Python:

            # If-Else-Anweisung
            x = 5
            y = 3
            
            if x > y:
              print("x ist größer als y")
            else:
              print("x ist nicht größer als y")
            
            # Schleife
            for i in range(5):
              print(i)  # Ausgabe: 0 1 2 3 4
            
            # While-Schleife
            i = 0
            while i < 5:
              print(i)
              i += 1  # Erhöhe i um 1
            
            # Funktionen
            def say_hello(name):
              print("Hallo, " + name + "!")
            
            say_hello("John")  # Ausgabe: Hallo, John!
